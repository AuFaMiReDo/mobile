### Deskripsi Tugas
Aplikasi ini dikembangkan menggunakan **Jetpack Compose** dengan fitur sebagai berikut:
1. **TextInput**: Field untuk menginputkan nama pengguna.
2. **Tombol Sapa**: Tombol untuk memproses nama yang diinput.
3. **TextView (Greeting)**: Menampilkan teks "Hello nama!" setelah tombol ditekan.
4. **Switch**: Fitur untuk mengubah tema aplikasi antara Mode Terang dan Mode Gelap secara dinamis.

### Cara Menjalankan Aplikasi
1. Buka **Android Studio**.
2. Pilih **Open** dan arahkan ke folder project.
3. Tunggu proses **Gradle Sync** selesai.
4. Hubungkan device fisik atau jalankan emulator, kalau saya disini menggunakan usb untuk terhubung dengan hp.
5. Klik ikon **Run 'app'** (segitiga hijau) di bagian toolbar atas.
